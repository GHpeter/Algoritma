```
 * 输出: 2

 * <p>

 * a, b 均可能是负数或 0
 ```

 ```
public class 剑指Offer65_不用加减乘除做加法 {

    public static void main(String[] args) {
        add(20,-10);
    }

    public static int add(int a, int b) {
        while (b != 0) {
            int c = (a & b) << 1;
            a ^= b;
            b = c;
        }
        System.out.println(a);
        return a;
    }
}
```
