#### 独一无二的出现次数

```
/**
 * 1207. 独一无二的出现次数
 * 给你一个整数数组 arr，请你帮忙统计数组中每个数的出现次数。
 * <p>
 * 如果每个数的出现次数都是独一无二的，就返回 true；否则返回 false。
 * <p>
 * <p>
 * <p>
 * 示例 1：
 * <p>
 * 输入：arr = [1,2,2,1,1,3]
 * 输出：true
 * 解释：在该数组中，1 出现了 3 次，2 出现了 2 次，3 只出现了 1 次。没有两个数的出现次数相同。
 * 示例 2：
 * <p>
 * 输入：arr = [1,2]
 * 输出：false
 * 示例 3：
 * <p>
 * 输入：arr = [-3,0,1,-3,1,1,1,-3,10,0]
 * 输出：true
 * <p>
 * <p>
 * 提示：
 * <p>
 * 1 <= arr.length <= 1000
 * -1000 <= arr[i] <= 1000
 */
 
 ```
 ```
public class 独一无二的出现次数 {
    public static void main(String[] args) {
        int[] nums = new int[]{-3,0,1,-3,1,1,1,-3,10,0};
        uniqueOccurrences(nums);
    }

    public static boolean uniqueOccurrences(int arr[]) {
        Map<Integer, Integer> arrMap = new HashMap<>();
        for (int x : arr) {
            arrMap.put(x, arrMap.getOrDefault(x, 0) + 1);
        }

        Set<Integer> timeSet = new HashSet<>();
        for (Map.Entry<Integer, Integer> key : arrMap.entrySet()) {
            timeSet.add(key.getValue());
        }
        boolean result = arrMap.size() == timeSet.size();
        System.out.printf("比对结果:%s%n", result);
        return result;
    }
}
```
